Model > Recommend > Collaborative Filtering

Predict product ratings using Collaborative Filtering

To estimate recommendations using Collaborative Filtering select a user id, a product id, one or more products to generate recommendations for, and product ratings. To generate recommendasions press the Estimate button or CTRL-enter (CMD-enter on mac).

Report > Rmd

Add code to Report > Rmd to (re)create the analysis by clicking the icon on the bottom left of your screen or by pressing ALT-enter on your keyboard.

If a plot was created it can be customized using ggplot2 commands (e.g., plot(result) + labs(caption = "Based on data from ...")). See Data > Visualize for details.
